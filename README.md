Στις 3D υλοποιήσεις τα datasets είναι συνθετικά, οπότε δεν χρειάζεται κάποια αλλαγή. Σε περίπτωση που επιθυμείτε την προσθήκη επιπλέον σημείων ανατρέξτε στη γραμμή 468 και για τα δύο 3D αρχεία.
